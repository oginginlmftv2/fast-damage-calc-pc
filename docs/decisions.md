# 決定事項

## アプリ基本情報
- アプリ名: 最速ダメ計 for PC
- 対象プラットフォーム: Android（iOS は将来対応）
- 価格モデル: 広告あり無料
- パーティ保存数: 10個

## v1.0 スコープ
### 実装する
- ダメージ計算
- パーティ保存（10個）
- レギュレーション切替
- ポケモン検索
- 技検索
- 特性
- 持ち物
- 性格
- 努力値
- テラスタイプ
- 確定数表示

### 実装しない
- OCR
- オーバーレイ
- 使用率ランキング
- 広告（v1.0では除外）
- ログイン
- クラウド同期
- SNS共有

## ゲーム仕様
- レベル固定: 50（レベルという概念なし）
- ダメージ計算式: 本編準拠
- レギュレーション: 約1ヶ月周期で更新

## チーム役割
- ChatGPT（チャッピー）: アーキテクト・ダメージ計算エンジン・全体設計
- Claude: UI実装（Flutter）
- Gemini: データ収集（pokemon.json, moves.json等）
- ユーザー（ぎん）: プロデューサー
